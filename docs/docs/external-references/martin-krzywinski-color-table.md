# Martin Krzywinski Color Table

[Martin Krzywinski's Color Name Database](https://mk.bcgsc.ca/colornames/) is a comprehensive resource containing **9,284 named colors** with conversions across multiple color spaces.

## Overview

This color table combines multiple color name lists into a single centralized resource, including colors from:

- bang-v2
- beer SRM chart
- colorhexa
- crayola
- n3
- ntc
- pantone
- raveling
- resene
- rgb.txt (X11)
- steve.hollasch
- wikipedia
- xkcd

## Features

- **9,284 named colors** with RGB coordinates
- **Multi-space conversions**: RGB, HSV, XYZ, Lab, LCH, CMYK
- **Color space conversion tables** for all even (r,g,b) combinations (2,097,152 colors)
- **Nearest neighbor analysis** using Î”E in Lab space for color distance
- **Swatch visualization** sorted by perceptual hue, brightness, and chroma

## Downloads

The website provides downloadable resources:

- Color name and space conversion table for all named colors
- Brief list with RGB components and names only
- Archive of parsed and tidied color lists
- Complete conversion table for 2M RGB colors (215 MB)

## Usage

This resource can be particularly useful when:

- Looking up standardized color names
- Working with color matching and similarity
- Understanding perceptual color relationships
- Implementing named color registries
- Researching color space conversions

Visit the [official website](https://mk.bcgsc.ca/colornames/) to explore the full database and download the color tables.
