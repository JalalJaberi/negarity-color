# Negarity Color Documentation

This directory contains the documentation website for [Negarity Color](https://github.com/jalaljaberi/negarity-color), built with [Docusaurus](https://docusaurus.io/).

## Local Development

```bash
yarn install
yarn start
```

This starts a local development server at `http://localhost:3000`.

## Build

```bash
yarn build
```

Builds the static site into the `build` directory.

## Deployment

This documentation will be deployed to GitHub Pages. The deployment process will be configured in the repository settings.
